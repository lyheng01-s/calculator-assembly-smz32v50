# Calculator using Assembly Language

This project is a simple assembly calculator for the SMZ32V50 microcontroller.

## Features

- Performs addition from 0 to 99
- Performs subtraction from 0 to 99
- Displays input and output using a 7-segment display
- Designed for microcontroller architecture practice

## Tech Stack

- SMZ32V50
- Assembly Language

## Files

| File | Description |
|---|---|
| `Project Final.ASM` | Main assembly source code |
| `sms32v50.INI` | Configuration file for the simulator/microcontroller environment |
